DimoInfois a Shellscript to perform Information Gathering 

### Features

- Check Website info
- Check Phone info
- IP Tracker
- Check Valid E-mail
- Check if site is Up/Down
- Check internet speed
- Check Personal info
- Find IP behind Cloudflare
- Find Subdomains
- Port Scan (Multi-threaded)
- Check CMS
- Check DNS leaking


### Usage:
```
git clone https://github.com/dimo5991/dimoinfo.git
cd dimoinfo
chmod +x dimoinfo.sh
bash dimoinfo.sh
```
